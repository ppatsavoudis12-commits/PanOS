
# PanOS 🖥️

A lightweight Windows utility tool built with Python.

## Features

- 🏠 **Home Dashboard** — Live CPU, RAM and DISK usage monitor
- 📝 **Notes** — Take and save notes in a floating window
- ⚙️ **CPU Panel** — Detailed system resource monitor
- 🖼️ **Wallpaper** — Change and manage your wallpaper themes
- 🎨 **Settings** — Switch between Dark, Neon and Blue themes
- 🔗 **Windows Shortcuts** — Quick access to built-in Windows tools like Task Manager and more
- ℹ️ **About** — App information

## Download

Head to the [Releases](https://github.com/ppatsavoudis12-commits/PanOS.git) page and download `PanOS.exe`

No Python or installation required — just double-click and run!

## How to Use

1. Download `PanOS.exe`
2. Double-click to open
3. Use the sidebar to navigate between features
